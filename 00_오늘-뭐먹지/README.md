# FE 스터디 프로젝트: 오늘-뭐먹지

> [노마드 코더의 바닐라 JS로 크롬 앱 강의](https://nomadcoders.co/javascript-for-beginners)를 수강 후 만든 시간에 따른 랜덤 음식 추천 사이트

<br>

## 1. 프로젝트 소개

#### 시간대별 랜덤 음식 추천

- 시간대별 추천 메뉴와 문구 수정한다.

  - ~ 오전 10시: `추천하는 아침 메뉴...`
  - 오전 11시 ~ 오후 14시: `추천하는 점심 메뉴는...`

  - 오후 15시 ~ 오후 17시: `추천하는 간식은...`
  - 오후 18시 ~ 오후 20시: `추천하는 저녁 메뉴는...`
  - 오후 22시 ~ 오전 9시: `추천하는 야식 메뉴는...`

- 시간 대별 메뉴 리스트가 다르게 설정되어 있다. 

<br>

#### 찜목록

- 찜목록에는 찜한 메뉴가 등록되어 있다.
- `찜목록`을 클릭하면 찜목록 Modal이 켜진다.
- Modal 내에 있는 `X`버튼을 클릭하면 찜목록에 메뉴가 사라진다. 

<br>

#### 로딩 창 

- 움직이는 이미지를 통해 동적인 효과 부여한다.
- 랜덤으로 음식관련 명언을 제공한다. 

<br>

#### 결과 창

- 랜덤 메뉴가 나온 뒤 `찜하기`를 누르면 추천 메뉴를 찜하기 목록에 추가할 수 있다. 
- 랜덤 메뉴가 마음에 들지 않으면 `다른 음식 먹을래...`를 클릭하여 다시 메뉴를 추천받을 수 있다. 
- `♥홈으로♥`를 클릭하면 홈으로 들어가진다.

<br>

## 2. 아쉬운 점

#### 코드 가독성이 떨어진다. 

- 처음 웹사이트 구현을 위한 코드를 작성하다보니, 문제가 많다.

- JS 파일을 나누지 않고, 하나의 파일에 다 몰아서 작성하다보니 코드 가독성이 떨어진다. 
- 다음에 진행할 때는 파일을 기능 별로 나누고 작성해야 할 것 같다.
